Periodic Table Elements of API

How to Use the API

This API provides information about elements on the periodic table. Below are the available endpoints:

Get All Elements

Endpoint: /api/elements

Sends a GET request to retrieve a JSON object containing an array of all elements.

Get Specific Element

Endpoint: /api/elements/{id}

Send a GET request where {id} is the element's name, symbol, or atomic number to retrieve a JSON object representing a specific element. For example, "Hydrogen" or "Gold" for names, "H" or "Au" for symbols, and 1 or 79 for atomic numbers. There is no need for you to specify which property you're using as the ID since the API will automatically determine this. The ID is also not case sensitive.

Response Structure

The information returned from the API includes:

  • Name
  • Symbol
  • Atomic Number
  • Atomic Mass
  • Standard State
  • Melting Point (Kelvin)
  • Boiling Point (Kelvin)
  • Subcategory (Metal, Nonmetal, Noble Gas, etc)
